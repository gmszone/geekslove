##教你设计物联网

这是一本书，确切地说是电子书《一步步搭建物联网系统》的APP版，并且我们在其中提供了实验室功能，当前允许模拟简单的http请求。


设计物联网系统是件有意思的事，它需要考虑到软件、硬件、通讯等多个方面。通过探索不同的语言、不同的框架，从而形成不同的解决方案。

在这里，我们将向您介绍物联网系统设计，并将带领您一步步搭建物联网系统。
